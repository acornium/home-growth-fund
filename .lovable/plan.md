

# Брендбук «Группа Смарт» — Интерактивный веб-сайт

## Концепция
Многостраничный интерактивный брендбук в строгом корпоративном стиле (тёмно-синий + белый), с навигацией по разделам. Сайт будет служить единым руководством по фирменному стилю компании.

---

## Структура и разделы

### 1. Главная страница / Обложка
- Полноэкранная заставка с логотипом «Группа Смарт»
- Краткое описание компании: частный инвестор в недвижимость малых форматов
- Навигация по разделам брендбука

### 2. Миссия и ценности
- Миссия компании — инвестиции в развитие недвижимости малых форматов в России
- Ключевые ценности бренда (надёжность, стратегический подход, развитие)
- Позиционирование: производственно-складская недвижимость, индустриальные парки, земельные участки, торговая недвижимость
- Tone of voice — деловой, уверенный, экспертный

### 3. Логотип
- Основная версия логотипа (загруженное изображение)
- Варианты: на белом фоне, на тёмном фоне
- Охранное поле (минимальные отступы)
- Минимальный размер
- Недопустимое использование логотипа (примеры: растягивание, изменение цвета, поворот)

### 4. Цветовая палитра
- Основной цвет: тёмно-синий (как на логотипе)
- Дополнительный: белый
- Вспомогательные оттенки: светло-серый для фонов
- Для каждого цвета: HEX, RGB, CMYK коды
- Интерактивное копирование кодов по клику

### 5. Типографика
- Основной шрифт для заголовков
- Шрифт для основного текста
- Иерархия: размеры, начертания, межстрочные интервалы
- Примеры использования в контексте

### 6. Фирменные носители
- Визитная карточка (лицо и оборот)
- Фирменный бланк / письмо
- Конверт
- Подпись для электронной почты
- Презентация (обложка слайда)

## Дизайн и UX
- Боковая навигация для переключения между разделами
- Плавные анимации при переходах
- Адаптивный дизайн (десктоп и мобильные устройства)
- Строгий минималистичный стиль, соответствующий инвестиционной тематике
